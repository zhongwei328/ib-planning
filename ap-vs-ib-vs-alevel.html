决策树(9 步)

  1. 目标国家是? — 美/英/加/港/其他
  2. 目标大学类型是? — 顶尖研究型 / 顶尖文理 / 综合 / 公立旗舰
  3. 孩子写作 / 综合能力强吗? — IB / A-Level / AP
  4. 想换美本学分吗? — AP 4-5 分换 6-10 学分
  5. 学校开什么课? — IB / AP / A-Level 决定可行
  6. 学校有 IB EE / TOK 老师? — 必备
  7. 孩子是全才还是专才? — IB 适合全才 / A-Level 适合专才
  8. G7-G8 校内 GPA 趋势? — 强 → IB HL 6-7 6-7 6-7 / 弱 → AP 自由选
  9. 预算考虑? — 全部差不太多(校均 15-30 万 / 年)
Henry 案例:
  • 目标:港三全奖 + 美本 Top 30 双路径
  • 强项:数学顶尖 (3 学期稳定 7+) + 1 真强项
  • 已选:IB MYP(因为学校 KCISEC 唯一课程)
  • 建议:IB DP 选课 = Math AA HL + Physics HL + Chem HL + 中文 / 英文 / 1 选修(SL)
  • 附加:1-2 科 AP(Calculus BC / Physics C / Chem)冲美本 TOP 30 换学分

结论

IB / AP / A-Level **没有"最好"** — 只有"最适合":

  • ✅ IB = 全面 + 美本偏好 + 工作量大
  • ✅ AP = 灵活 + 换学分 + 美本偏好
  • ✅ A-Level = 专才 + 英联邦必备 + 选课少
