TOEFL MyBest 拼分政策(2019 起)

拼分核心:取最近 2 年内 4 次考试中,4 个单科(读 / 听 / 说 / 写)的**最高分**重新组合。
例子:
  • 第 1 次:读 28 / 听 27 / 说 22 / 写 25 = 102
  • 第 2 次:读 28 / 听 28 / 说 24 / 写 26 = 106
  • 第 3 次:读 29 / 听 28 / 说 23 / 写 27 = 107
  • 第 4 次:读 28 / 听 28 / 说 25 / 写 25 = 106
MyBest:读 29 / 听 28 / 说 25 / 写 27 = 109(单次最高 107 + 拼分 +2)

SAT 拼分政策(2024 起)

Single-Section Superscore:取多次考试中**各 section 最高分**重新组合。
SAT 2 sections:EBRW (800) + Math (800) = 1600
例子:
  • 第 1 次:EBRW 700 / Math 750 = 1450
  • 第 2 次:EBRW 730 / Math 720 = 1450
Superscore:EBRW 730 / Math 750 = 1480
